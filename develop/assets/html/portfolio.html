Music Spider

A music searching application created by a small team of developers in training. Music Spider is a library-expanding music search tool that allows the user to find artists by song lyrics and recommends similar artists for the user to listen to.

Currently learning: Node
Day Planner

This app allows the user to save their daily schedule (business hours) into individual time slots which are color coded according to whether they are past, present, or future. The clock can be toggled between twelce and twenty-four hour mode.

Weather Dashboard

A weather forecast application that allows the user to search current and future weather conditions by city. The searches are saved and added to a sidebar so the user can view the same cities again at the click of a button.
