A music searching application created by a small team of developers in training. Music Spider is a library-expanding music search tool that allows the user to find artists by song lyrics and recommends similar artists for the user to listen to.

Day Planner

A day planner that allows the user to save their daily business schedule. Saved events are set in local storage. Time blocks are arranged in hour long blocks and are color coded according to past, present, and future times. The clock can be toggled between 12 and 24-hour modes.

Weather Dashboard

A weather application that allows the user to search current and future weather conditions by city. The current conditions and five-day forecast of the searched city will be appended to the page, and the searched city will be saved to local storage and assigned a sidebar button so that the city weather can be checked again easily.
